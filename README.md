# React + Vite

**App.js:** Es el componente principal que mantiene el estado global de las notas y el filtro de prioridad.
**NoteForm.js:** Maneja el formulario para agregar nuevas notas. Cada nota tiene un texto y un nivel de prioridad.
**NoteList.js:** Recibe las notas filtradas y las renderiza usando el componente NoteItem.
**NoteItem.js:** Muestra una nota individual con un botón para eliminarla.
**NoteFilter.js:** Componente para filtrar las notas según su prioridad.
**PropTypes:** Se ha implementado la validación de props usando PropTypes para garantizar que los componentes reciban los datos en el formato correcto.
